# Customer-Relationship-Management
Single Page Application (SPA) infographic that serves as a plain-English tutorial for Basic CRM Administration.
CRM Administration 101
A visual guide to managing, configuring, and deploying your business's most valuable asset: Customer Relationships.
This infographic breaks down the complex topics of management, configuration, and deployment into digestible visual sections using a "Digital Ocean" color palette (Deep Blues and Vibrant Cyans) to maintain a professional yet engaging tone.

Generated Files:

index.html: A self-contained HTML file containing the tutorial content, Chart.js visualizations, Tailwind CSS styling, and interactive elements.

Key Features of the Infographic:

The Gatekeepers (User Permissions): Visualized with a Donut Chart to show the hierarchy of access roles.

The Toolkit (Asset Management): A Horizontal Bar Chart breaking down the types of assets an admin manages.

The Launch Cycle (Deployment): A custom HTML/CSS Timeline (No SVGs) illustrating the safe deployment of changes.

The Daily Grind (Responsibilities): A Radar Chart visualizing the balance of skills required (Tech vs. Soft Skills).

Interactive Elements: All charts are interactive with custom tooltips handling multi-line text.
